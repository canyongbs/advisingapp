# #关键AI + CRM功能
# # #人工智能

我们强大的个人助理旨在为您的学生服务专业人员的日常工作提供支持。 助理可以帮助的一些关键功能包括：

-研究任务
-沟通草稿
-语言翻译
-内容创作
-学生档案分析
-项目规划
-构思
-还有更多...

# # #学生服务CRM
我们的学生服务CRM旨在支持未来和入学学生的管理。 一些主要功能包括：

-记录管理
-电子邮件和短信
-服务管理
-人口细分
-任务管理
-互动跟踪
-文件和文档
-还有更多...

# #贡献

该项目的软件由Canyon GBS Inc.内部的专业工程团队创建和管理。\
\
所有对项目的贡献都必须经过预先批准，以确保产品的完整性。使用正确的“更改类型”标签创建问题，并包括您提议的更改的详细信息。Canyon GBS的产品团队成员将进行审核，如果获得批准，您可以分叉回购并创建拉取请求。完成后， Canyon GBS的工程师将在合并PR之前审查质量保证。您可能会被要求进行更改，以满足我们的专业人员质量标准
1.分叉项目
2.创建功能分支（ `git checkout -b feature/AmazingFeature` ）
3.提交您的更改（ `git commit -m'添加一些AmazingFeature '` ）
4.推送到分支（ `git push origin feature/AmazingFeature` ）
5.打开拉取请求

打开拉取请求时，请确保您符合以下要求：
-为公关添加工单/问题编号，并简要说明所做的更改。 或者，如果没有工单/问题，请在公关标题中详细说明所做的更改
-关联相关工单或问题，或描述此PR解决/实施的问题/功能。
-解决了所有冲突（如有）。
-在打开公关之前，请务必在最新的上游“主”分支机构之上重新建立分支机构公关。
